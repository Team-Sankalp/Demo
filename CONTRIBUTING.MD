# Contributing to Demo

## Branch Naming
- Frontend: frontend/feature-name
- Backend: backend/feature-name
- Database: database/feature-name
- AI: ai/feature-name

## Commits
- Use simple messages: feat(frontend): add login
- Keep it short and clear
